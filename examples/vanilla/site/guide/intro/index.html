Guide Introduction

This page explains indexing, tokenization, and scoring. Tokens in titles receive the highest weight, headings receive medium weight, and body tokens receive base weight.

Tokenization

Search Kit lowercases text, splits on non-alphanumeric characters, keeps only letters and numbers, drops short tokens, and removes common stopwords.

Scoring

Title matches add five points. Heading matches add three points. Body matches add one point. A small bonus is applied when multiple query terms match the same document.
